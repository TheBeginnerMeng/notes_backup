# 整理的考点知识

- 静态语言：编译执行；脚本语言：解释执行，JS/PHP/Python

- IPO：Input、Process、Output
- Python保留字keyword有35个
- 特点：平台无关，强制可读，支持中文，生态高产

- 十六进制：hex 0x3F2，八进制：octal 0o1762，二进制：0b001111
- 浮点类型必须带有小数部分（可以是0）
- 浮点数有2种表示方法：十进制形式的一般表示和科学计数法表示，除十进制外，浮点数没有其他进制表示形式
- 复数可以看作是二元有序实数对(a, b)，表示：a + bj
- divmod(x,y)：(x//y, x%y) # 元组
- 格式控制标记包括：`<填充><对齐><宽度>,<.精度><类型>`6个字段，如`{:*>10,2f}` d:十进制，b: 二进制，o:八进制，x: 十六进制
- chr(x):Unicode编码x —-> 单字符，ord(x)：单字符x —> Unicode编码
- str.split(sep=None)，返回一个列表
- str.replace(old, new)，返回字符串str的副本
- str.strip(chars)，从str中去掉左右两侧chars中的每一个字符，返回处理后的str
- str.join(iter)，从iter的每一个元素后添加一个str
- Python循环结构：遍历循环，无限循环
- def func(非可选参数，可选参数)
- 组合数据类型：序列类型：str/tuple/list，集合类型：set，映射类型：map
- list.insert(i, x)：在list第i位置增加元素x
- list.pop(i)：删除第i项元素
- list.copy()，生成一个新列表，复制list的所有元素
- dict.get(key, default)
- r：只读，w：覆盖写，x：创建写，a：追加写，b：二进制文件模式，t：文本文件模式，+：与r/w/x/a一同使用，增加读写功能
- f.readlines()：返回一个列表，每行内容一个元素
- f.write(s)：写入一个字符串，f.writelines(lines)：写入元素为字符串的列表
- turtle.fd(distance)，当前方向出发distance
- turtle.right/left(angle)，向右/左旋转angle角度
- turtle.seth(angle)，0-east，90-north，180-west，270-south
- turtle.circle(radius, extent=None)
- random.seed(a=None)
- random.random()，生成[0.0, 1.0)之间的随机小数
- random.randint(a, b)，生成[a, b]之间的整数
- random.sample(pop, k)，从pop中随机选取k个元素，以列表返回
- PyInstaller -F xxx.py，打包生成一个可执行文件，保存在dist中，- - path的作用是指定代码所依赖非标准库的路径
- jieba.lcut(“xxxxx”)，中文切词，返回列表
- 算法的4个基本特定：可行性，确定性，有穷性，拥有足够的情报
- 二叉树第k层上，最多有2^(k-1)个结点
- 深度为m的二叉树，最多有2^m-1个结点
- 二叉树中，度为0的点总是比度为2的结点多一个
- 交换类排序：冒泡（最坏，n(n-1)/2），快排（最坏，n(n-1)/2）
- 插入类排序：简单插入(最坏，n(n-1)/2)，希尔O(最坏，n^1.5)
- 选择类排序：简单选择(最坏，n(n-1)/2)，堆排序O(最坏，nlog2n)
- 数据库技术的目标：解决数据共享问题，数据系统的核心是数据库管理系统
- 数据库系统的三级模式：概念模式：是数据库系统中全局数据逻辑结构的描述，一个数据库只有一个概念模式
- 外模式：一个概念模式可以有若干外模式，数据库用户能够看见和使用的局部数据的逻辑结构和特征的描述
- 内模式：物理模式，数据库物理存储结构和物理存取方法
- 软件设计原则：抽象，模块化，信息隐蔽，模块独立性
- 结构化程序设计原则：自顶向下，逐步求精，模块化，限制使用goto
- 结构化分析方法常用工具：数据流图DFD（需求分析阶段），数据字典DD，判定表，判定树
- 软件设计包括：结构设计，数据设计，接口设计，过程设计
- 软件需求规格说明书（SRS，Software Requirement Specification）是需求分析阶段的最后成果，是软件开发中的重要文档之一。它有以下几个方面的作用：① 便于用户、开发人员进行理解和交流；② 反映出用户问题的结构，可以作为软件开发工作的基础和依据；③ 作为确认测试和验收的依据。
- 关系数据库管理系统能实现的专门关系运算，包括选择运算、投影运算、连接运算。
