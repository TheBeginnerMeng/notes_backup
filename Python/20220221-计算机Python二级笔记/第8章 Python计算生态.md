## 8.2 程序设计方法论

### 8.2.1 自顶向下设计

- 一个解决复杂问题行之有效的方法是自顶向下的设计方法

整个过程分为四个步骤：

- 将算法表达为一系列小问题
- 为每个小问题设计接口
- 通过将算法表达为接口关联的多个问题来细化算法
- 为每个小问题重复上述过程
